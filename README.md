📖 Project Overview

This project is a Java-based Payroll System developed as part of the MO-IT101 course. The application is designed to compute employee salaries, including government deductions and net pay, using a structured and modular programming approach.

The system demonstrates core programming concepts such as:
-Object-oriented design
-Method decomposition
-Input/output handling
-Algorithmic problem solving

🎯 Objectives

The main objectives of this project are to:

-Accurately compute employee gross pay, deductions, and net salary
-Apply proper modular programming techniques
-Improve code readability, structure, and maintainability
-Simulate a real-world payroll computation system

⚙️ Features

✅ Semi-monthly salary computation (15th and 30th payout)

✅ Automated calculation of government deductions:

-SSS
-PhilHealth
-Pag-IBIG

✅ Net pay computation

✅ User input handling via console

✅ Organized and reusable methods for each calculation

✅ Clear output formatting for payroll summary
